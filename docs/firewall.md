# Windows Firewall

## Objetivo

Verificar se o Windows Firewall está habilitado em todos os perfis de rede.

## Comando executado

```powershell
Get-NetFirewallProfile
```

## Resultado

_Preencher após executar o comando._

## Observações

O Windows Firewall ajuda a bloquear conexões não autorizadas e é uma das primeiras camadas de proteção de um sistema Windows.
